# Datastory-generaties-in-conflict-python
# Datastory – Generaties in Conflict (Python)

Deze repository bevat de Python-scripts die zijn gebruikt voor de dataverwerking en visualisaties van de datastory *Generaties in Conflict*, ontwikkeld binnen de minor **Big Data & Design**.

## Inhoud van de repository

De volgende scripts/proeven zijn opgenomen:

- **A.1** Testdiagram: koffiekopje en boontjes (verkennende visualisatie)
- **A.1.1** Treemap-grafiek
- **A.2** Productiviteitsproef 1
- **A.2.1** Grafiek proef 2 – technologie
- **A.3** Gemiddelde pilots van 4 Day Week Global
- **A.4** Grafiek proef 5 – werkvormen

Deze scripts laten zien hoe data is opgeschoond, geanalyseerd en vertaald naar visuele representaties met behulp van Python.

## Data (CSV-bestanden)

De gebruikte data is grotendeels handmatig samengesteld op basis van publieke databronnen, waaronder NEA (TNO & CBS), RIVM, OECD, Eurostat en 4 Day Week Global.  
Waar mogelijk zijn deze datasets opgeslagen in CSV-bestanden en gebruikt als input voor de scripts.

Indien niet alle oorspronkelijke CSV-bestanden beschikbaar zijn, tonen de scripts alsnog de datastructuur, bewerkingen en visualisatielogica.

## Techniek

- Python 3
- pandas
- Plotly (en/of Matplotlib)

## Context

Deze repository vormt de technische onderbouwing van **Blok 1** van de minor *Big Data & Design*.  
De methodologische keuzes, interpretatie van data en reflectie zijn uitgewerkt in het bijbehorende verslag en de bijlagen.
